Tools I use

Yes I am an Unix dork. :)

  1. Mutt - an email client
  2. Irssi - a chat client
  3. Vim - an editor, XML support sucks
  4. Python - a programming language
  5. Screen - for maintaining a session which I can disconnect and reconnect to
  6. Ion - window manager
  7. Mozilla - For browsing
  8. Rss2email - For those news feeds
  9. Uxterm - Unicode terminal of choice
  10. Bash - I might change to something that does history better

All available in excellent Debian packages. I am a big fan of the console as you may or not notice. I use the console when possible because it is fast and functional. Simply put, it gets the job done quickly. So I can do something else.
