The VI iMproved editor

Introduction

In order to use a computer, you need to learn the skills to use an editor. Out of the zillions out there, I have settled down with vim. Namely because I like to use the console for numerous reasons (screen, pine, irssi), and I want a fast powerful editor. I have used emacs before, but I switched because:

  • New version of emacs with GTK and variable font size freaked me out
  • I could never ever get my .emacs just right! :/
  • It is a little slow at times
  • The console version simply sucks
  • Different versions and branches freaks me out
  • Default undo frankly sucks
  • Bit overwhelming for me...

Getting started

If you are running linux like a good person, you will have vim or at least vi on your system. That is one of the great aspects of vim is that is it everywhere. Vim is the improved version of vi btw. I recommend you launch the vimtutor program to get to know the basics of vim. Although the F1 help in vim is very good. I use it all the time (CTRL-t to get back). What takes getting used to in vim is the different modes. It sucks at first, I know, however you will learn after awhile it is not so bad. Just be patient, and do not lose your cool. =)

My reference

As a mechanism to get me to learn vim better, I have not only made this page for you but for me to list some key shortcuts I tend to forget.

CTRL-w _
Max window
CTRL-w CTRL-w
Swap Windows
CTRL-w c
Closes the current window
CTRL-w o
Closes all the other windows
CTRL-w r
Rotate windows
